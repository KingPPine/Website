Minecraft server.

Goal was to have a Minecraft Server
for me and Matt to play 24/7

Started renting an Amazon EC2 Medium instance

Downloaded a key and SSH'd into the server

Sudo chmod to grant appropriate permissions

sudo yum install to install & update java, along with other modules

Added a script in the startup folder to boot minecraft automatically on server start
This is in var/lib/cloud/scripts/per-boot/

Set up weekly backups for the instance

Costs would be $30 a month, so I set up a free micro EC2 instance to launch / shut it down.
